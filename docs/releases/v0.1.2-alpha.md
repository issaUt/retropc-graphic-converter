更新内容:

- D88生成時の既定動作を「新規作成」に変更
- `既存D88ファイルがある場合はファイルを追加` チェックを追加

## Notes

- このZIPにはGUIのみ含まれます

- 初回起動後、`Settings` タブで Ruby と Script のパスを指定してください
- 対応する Ruby core は `pngconvMZ 0.1.3` です
