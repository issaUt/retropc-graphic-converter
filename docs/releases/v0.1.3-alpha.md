# RetroPC Graphic Converter GUI 0.1.3-alpha

更新内容:

- `Machine` セレクタを追加し、`MZ-2500` / `MZ-2861` を切り替え可能に
- 機種ごとに `Mode` / `Layout` の候補を切り替えるように更新
- `MZ-2500` の 512色モードから `640x200` を除外
- `MZ-2861` では `8 / 16 / 4096` のみを表示し、`320x200` と `512` を無効化
- `Mode` コンボ幅を `Machine` に合わせて調整
- GUI版数を `0.1.3-alpha` に更新

## Notes

- このZIPにはGUIのみ含まれます
- Ruby本体と `pngconvMZ.rb` は同梱していません
- 初回起動後、`Settings` タブで Ruby と Script のパスを指定してください
- 対応する Ruby core は `pngconvMZ 0.1.4` です

Ruby core:

https://github.com/issaUt/mz-ruby-graphic-core

- `MzRubyConvGui-win-x64.zip`
