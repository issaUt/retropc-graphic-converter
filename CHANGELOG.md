# Changelog

## 0.1.3-alpha - 2026-05-06

- `Machine` セレクタを追加し、`MZ-2500` / `MZ-2861` を切り替え可能に
- 機種ごとに `Mode` / `Layout` の候補を切り替えるように更新
- `MZ-2500` の 512色モードから `640x200` を除外
- `MZ-2861` では `8 / 16 / 4096` のみを表示し、`320x200` と `512` を無効化
- `Mode` コンボ幅を `Machine` に合わせて調整
- GUI版数を `0.1.3-alpha` に更新

## 0.1.2-alpha - 2026-05-05

- D88生成時の既定動作を「新規作成」に変更
- `既存D88ファイルがある場合はファイルを追加` チェックを追加
- `BRD/BSDファイルを残す` チェックと組み合わせたD88運用を整理
- D88関連UIのレイアウトを調整

## 0.1.1-alpha - 2026-05-05

- D88ディスクイメージ生成UIを追加
- `BRD/BSDファイルを残す` チェックを追加
- D88内部ファイル名長の事前チェックを追加
- 512色モードのファイル命名を `_FR` / `_FG` / `_FB` に変更
- `split320x200` の結合BSD名を `_c.bas.bsd` に変更
- D88関連UIの調整と表示修正

## 0.1.0-alpha - 2026-04-23

- 初回公開版
- Rubyスクリプト設定とバージョン確認
- PNG/JPEG入力対応、各種変換オプションとプレビュー表示
- JSON / Log / Files 表示
